NotebookPro is a WinForms and .NET Desktop based lightweight notebook application created by EdvinasDev_98.

**Important: Runtime Setup**

**Before launching the app, navigate to the runtimes folder and extract the archive that matches your operating system:**

- Windows x64 → win-x64.zip
- Windows x86 → win-x86.zip
- Windows ARM64 → win-arm64.zip
- macOS / OS X → osx.zip
